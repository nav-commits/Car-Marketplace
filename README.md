## Overview
This project is a **frontend-only prototype** of a car marketplace homepage. It includes:

- Hero section with main title, subtitle, and filters.
- Responsive layout and mobile-friendly design.
- Components designed for reusability using **React + TypeScript**.
- Styling implemented with **Tailwind CSS**.
- Icons integrated via **Heroicons** and **React Icons**.

The project follows the Figma design linked above.

## Features
- Fully responsive homepage layout.
- Hero section with background image and overlay.
- Filter dropdowns for searching cars.
- Modern UI elements and clean typography.
- Easy to extend for other pages or dynamic data.

## Tech Stack
- **Next.js 16** – React framework for server-side rendering and static site generation.
- **TypeScript 5** – Type-safe JavaScript.
- **Tailwind CSS 4** – Utility-first CSS framework.
- **Heroicons / React Icons** – Icon libraries.
- **ESLint** – Code linting and formatting.
